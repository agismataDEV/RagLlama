In order for the programm to run , you should use an input file which is used for the parameters and constants.

Moreover a "matrixcontrastinput.bin" should be used. For the functionality of the program, if you change the name, 
you should change the input in ContrastGenFilt.F90 before doing the compilation of INCS.
This bin file is generated by a matlab file called ContrastGeneration.m and it includes information for the grid discretization.
In order to work, first it should be an executable file so in linux use :
	 chmod +x matrixcontrastinput.bin
